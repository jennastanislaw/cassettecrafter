Golden Gate Assembly Optimizer

This web application will optimize the order of specific plasmid cassettes for Golden Gate Assembly.

Created by: Jenna Stanislaw, Siobahn Kells, Anna Boerman, and Carolina Valderrama Hincapie

Affiliation: University of Colorado Boulder

Sequence Optimizer Instructions

  1. Enter your Backbone Sequence in the first input box.
  2. Enter your Protein Sequence in the second input box.
  3. Ensure each sequence contains only the letters a, g, t, and c (case insensitive).
  4. Click the "Optimize" button to process the sequences.
  5. Review the results displayed after optimization.
